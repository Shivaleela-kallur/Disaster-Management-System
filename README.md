# Disaster Management System

## Overview
The Disaster Management System is a Java-based application designed to help manage and coordinate disaster-related information efficiently. The system allows users to add, update, delete, and view disaster events and resources, facilitating better disaster response and resource allocation.

## Features
- Add, update, and delete disaster records.
- Manage resources and volunteers.
- Generate reports and alerts.
- User roles and access management.

## Technologies Used
- Java (OOP concepts)
- SQL (Database management)
- Eclipse IDE

## Project Structure
- src/ - Source code files.
- resources/ - Application resources and configuration files.
- settings/ - Eclipse project settings.
